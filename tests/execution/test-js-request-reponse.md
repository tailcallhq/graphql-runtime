# Js Request Response Hello World

#### server:

```graphql
schema @server @link(type: Script, src: "../http/scripts/test.js") {
  query: Query
}

type Query {
  hello: String @http(baseURL: "http://localhost:3000", path: "/hello")
  hi: String @http(baseURL: "http://localhost:3000", path: "/hi")
}
```
